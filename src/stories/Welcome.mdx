A modern typewriter animation for React — smart backspace, natural typing variance, punctuation pauses, imperative controls, and a fully headless `useTypewriter` hook. Styled with **Tailwind CSS v4**.

---

## Installation

```bash
npm install eglador-ui-react-typewriter
```

## Setup

Add the following to your global stylesheet (e.g. `app/globals.css`):

```css
@import "tailwindcss";
@source "../node_modules/eglador-ui-react-typewriter";
```

## Quick Start

```tsx
import { Typewriter } from "eglador-ui-react-typewriter";

<Typewriter deleteMode="smart" className="text-3xl font-semibold">
  <Typewriter.Item>Build with React.</Typewriter.Item>
  <Typewriter.Item>Style with Tailwind.</Typewriter.Item>
  <Typewriter.Item>Ship faster.</Typewriter.Item>
</Typewriter>
```

Each `Typewriter.Item` accepts its own `className` and `style`, plus per-item timing overrides:

```tsx
<Typewriter className="text-3xl font-semibold" deleteMode="smart">
  <Typewriter.Item className="text-blue-600">Build with React.</Typewriter.Item>
  <Typewriter.Item className="text-cyan-500">Style with Tailwind.</Typewriter.Item>
  <Typewriter.Item
    className="bg-gradient-to-r from-pink-500 to-violet-500 bg-clip-text text-transparent"
    typingSpeed={120}
    pauseDuration={3000}
  >
    Ship faster.
  </Typewriter.Item>
</Typewriter>
```

## Features

- **Compound API** — `<Typewriter>` + `<Typewriter.Item>` (Radix / shadcn pattern), per-item `className` / `style` / timing overrides
- **Multi-string rotation** — cycle through any number of items, with `loop` on/off
- **Smart backspace** — `deleteMode="smart"` keeps the common prefix with the next string and only retypes the differing suffix (Apple landing-page pattern)
- **Natural typing** — `variance` adds ± jitter to every character delay so the rhythm feels human; `punctuationDelays` adds extra pauses after `.` `,` `!` `?` etc.
- **Imperative controls** — `play()`, `pause()`, `reset()`, `skip()` exposed via `controlsRef`
- **Lifecycle callbacks** — `onType`, `onChange`, `onLoop`, `onComplete`
- **Cursor styles** — `line` / `block` / `underscore` with `smooth` / `hard` / `none` blink, or override with any custom character
- **Render-prop API** — full control over the wrapping element and surrounding markup
- **Headless hook** — `useTypewriter()` for completely custom UIs
- **Accessible** — `role="text"` + `aria-label` joins all strings so screen readers announce the full content once instead of character-by-character
- **Reduced-motion friendly** — honors `prefers-reduced-motion: reduce` by rendering the final string instantly with no cursor
- **SSR safe** — renders empty on the server, animation starts on client mount
- **Zero runtime dependencies** — only `clsx` + `tailwind-merge`, both pre-bundled
